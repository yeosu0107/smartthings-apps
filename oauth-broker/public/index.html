Seoul Bus Alarm — SmartThings 연결

아래 버튼으로 SmartThings 계정에 한 번 로그인하면 드라이버가 사용할 refresh token이 발급됩니다.

Connect to SmartThings

  1. 발급된 refresh token을 화면에서 복사합니다.
  2. SmartThings 앱 → Seoul Bus Stop 디바이스 → 설정 → Refresh Token 칸에 붙여 넣습니다.
  3. 같은 화면에서 Speaker Device ID(갤럭시 홈 미니)도 함께 입력합니다.

토큰은 이 사이트에 저장되지 않으며, 페이지를 닫으면 다시 볼 수 없습니다.
